# CloudLiteracy

**CloudLiteracy** is a full-stack educational platform designed to prepare learners for a career in DevOps through structured pre-DevOps coursework. It offers 7 learning modules, each containing PDF resources, video content, and interactive quizzes with a weighted progress scoring system (videos 40%, PDFs 30%, quizzes 30%). Access to modules is gated behind a payment system that supports multiple gateways — including Stripe (Visa/Mastercard), PayPal, MTN Mobile Money, and Orange Money — making it well-suited for its target audience in Cameroon. The platform is built on a **React** frontend and a **Node.js/Express** backend with **MongoDB** for data storage, JWT-based authentication, and real-time features powered by Socket.io.

Beyond core learning, CloudLiteracy includes a robust admin ecosystem with three tiers of access: regular users, admins, and a super admin. Admins can monitor user progress, view platform analytics, and track real-time online activity with geolocation. The super admin has exclusive capabilities such as managing other admins (creating, suspending, or reinstating them with real-time session revocation), and operating an **AWS Exam Vouchers system** — which supports encrypted voucher storage, bulk CSV/Excel uploads, lifecycle tracking (unused → assigned → redeemed → expired), and assignment to learners for 6 AWS certification exams. Additional features include a donation system, a 5-star rating system, password reset via email, CAPTCHA, and a sleek black-and-gold themed UI.
